# Default Install Location of OpenCV

This function returns the location at which OpenCV should be installed
by default.

## Usage

``` r
defaultOpenCVPath()
```

## Value

A character string.

## Author

Simon Garnier, <garnier@njit.edu>

## Examples

``` r
defaultOpenCVPath()
#> [1] "/home/runner/work/_temp/Library/ROpenCV"
```
